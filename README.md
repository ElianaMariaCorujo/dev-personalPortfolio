# Entorno de Desarrollo Automatizado con IA

Este repositorio inicia un proyecto web moderno y liviano para prototipado rapido, con foco en inclusion, accesibilidad y buenas practicas para equipos de desarrollo.

## Objetivo

Crear una base profesional para experimentar automatizaciones web asistidas por IA, facilitando iteraciones rapidas sobre interfaz, logica de formulario y experiencia de usuaries.

## Estructura inicial

- `index.html`: interfaz de bienvenida con formulario inclusivo y accesible.
- `styles.css`: estilos complementarios y ajustes de accesibilidad.
- `script.js`: comportamiento basico del formulario y mensajes de estado.
- `.cursorrules`: lineamientos de perfil profesional, inclusion y WCAG.

## Tecnologias

- HTML5 semantico.
- Tailwind CSS via CDN.
- JavaScript vanilla.

## Criterios de inclusion y accesibilidad

- Lenguaje inclusivo en placeholders, textos y comentarios.
- Campos opcionales para pronombres e identidad de genero no binaria.
- Uso de etiquetas semanticas, asociacion correcta entre `label` e `input`.
- Estados de foco visibles y soporte de navegacion por teclado.
- Base alineada a WCAG 2.1 nivel AA.

## Como ejecutar

1. Abrir `index.html` en el navegador.
2. Completar el formulario y validar el mensaje de estado.
3. Iterar sobre estilos y logica segun necesidades del proyecto.
